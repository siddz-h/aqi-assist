AQI Assist

AQI Assist is a simple web application that shows the Air Quality Index (AQI) of a city.
It helps users understand air quality levels and basic health impact.

Features

Search AQI by city name

Displays AQI value and category

Shows basic AI-like health advice

Clean chat-style interface

Technologies

HTML

CSS

JavaScript

How to Run

Open the project folder

Run index.html using Live Server or any local server

Enter a city name and click Send

Purpose

This project is created for learning and demonstration purposes.
